1. Открыть таблицу с учетом работ по проекту в рамках техподдержки
2. Перейти в пункт меню Инструменты -> Редактор скриптов...
3. Выбрать пункт "Пустой проект"
4. Вставить в окно редактирования код из файла по ссылке:
https://github.com/orlovrs/GDdocs/blob/master/TechSup/ReportActCount.txt
если вы подключаете скрипт к проекту, где НЕ нужна колонка "Категория" в отчете
https://github.com/orlovrs/GDdocs/blob/master/TechSup/ReportActCountWithCategory.txt
если нужна эта колонка
5. Сохранить скрипт под любым именев, выбрав пункт меню Файл -> Сохранить
Скрипт подключен, будет работать при обновлении самого файла.
